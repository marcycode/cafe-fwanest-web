# Café Fwanest

A bilingual, mobile-first website for Café Fwanest — a waterfront café and gelato destination in Gatineau, Québec.

## Live site

[cafe-fwanest.marcelvids.chatgpt.site](https://cafe-fwanest.marcelvids.chatgpt.site)

## Highlights

- French and English content switching
- Waterfront-first editorial design
- Responsive navigation and mobile directions CTA
- Interactive menu category filters
- Café, event, gallery, review, and visit sections
- Accessible labels and reduced-motion support
- Google Maps, Instagram, Facebook, and telephone links

## Tech stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Vinext and Cloudflare Workers

## Local development

Requires Node.js 22.13 or newer.

```bash
npm ci
npm run dev
```

Run the checks with:

```bash
npm run lint
npm test
```

## Project structure

- `app/page.tsx` — bilingual page content and interactions
- `app/globals.css` — responsive visual system and layouts
- `public/images/cafe/` — Café Fwanest photography
- `.openai/hosting.json` — Sites deployment configuration
